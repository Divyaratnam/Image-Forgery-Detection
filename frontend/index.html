📘 Project Overview

This project focuses on detecting digital image forgery using deep learning techniques. The system classifies images as forged or authentic.

  • GAN for data augmentation
  • Swin Transformer for hierarchical feature extraction
  • EfficientNet for optimized classification

📄 Project Report (Summary)

Digital image forgery has become a major challenge with the widespread availability of image editing tools. This system proposes a GAN-augmented Swin Transformer and EfficientNet architecture to enhance forgery detection accuracy.

The system preprocesses images, performs feature extraction, and classifies images efficiently. Experimental results show high accuracy and robustness against common forgery techniques such as copy-move and splicing.
